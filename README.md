Loan Decision Support System (Frontend)

Project Overview
This project is a Loan Decision Support System Frontend built using modern web technologies.
It provides a user-friendly interface where users can:

Register & Login
Apply for Loan
View Decision Status
See Confidence Score
Check Loan History
Manage Profile
View Dashboard Summary
The system is designed for high-stakes AI-based decision making, where transparency and explainability are important.

Tech Stack:
Next.js (App Router)
React
TypeScript
Tailwind CSS
Framer Motion
Modern UI Components


Features :
Authentication
  User Registration
  User Login

Loan Application
  Loan amount
  Employment details
  Income
  Credit score
  Debt-to-income ratio
  Down payment
  Monthly debts
 
Dashboard
  Decision status
  Confidence gauge
  AI explanation card
  
Summary overview
History
Previous loan applications
Status tracking

Profile
View user information
Update personal details


Explainable AI Concept
The system is designed keeping Explainable AI (XAI) principles in mind:
Displays decision status (Approved / Rejected)
Shows confidence score
Explains top positive & negative influencing factors
